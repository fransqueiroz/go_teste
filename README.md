# Teste Golang

Uma breve descrição sobre o que esse projeto faz e para quem ele é


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/fransqueiroz/go_teste
```

Entre no diretório do projeto

```bash
  cd go_teste
```

Inicie a aplicação com

```bash
  docker-compose up --build
```
